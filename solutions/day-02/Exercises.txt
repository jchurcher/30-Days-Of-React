Exercises: What is React?

    What is React?
        React is a JS library for building single page web apps dynamically.

    What is a library?
        A library is a piece of prewritten code that you can import into your own project to use, it saves you time and effort rewriting the same code again.

    What is a single page application?
        A web application that does not redirect you to a different web page or url, the page is built dynamically in the browser without having to request another html file.
